  <action>
    Implement `run_attach` for Windows:
    1. Resolve session ID and verify its status is Running.
    2. Read and print the tail of `~/.nono/sessions/{id}.log` for scrollback.
    3. Connect to the Data Pipe `\\.\pipe\nono-data-{id}`.
    4. Implement bi-directional streaming:
       - Pipe output (from ConPTY) to local stdout.
       - Local stdin to Pipe input (to ConPTY).
    5. Handle the detach escape sequence (Ctrl-] d).
    6. Ensure clean exit when the agent session terminates.
  </action>
