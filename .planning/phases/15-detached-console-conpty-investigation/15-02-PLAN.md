<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| DETACHED_PROCESS supervisor → grandchild | Token shape determines grandchild's access-check surface; changing token changes security posture |
| supervisor → WFP service | Session SID (direction-a) or AppID (direction-b) used as network-filter identity |
| detached stdout pipe → nono attach client | Pipe carries child stdout; direction-b only; no auth on pipe content |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-15-02-01 | Elevation of Privilege | direction-b: null token removes WRITE_RESTRICTED access check from grandchild | accept | Job Object containment + filesystem sandbox (CapabilitySet) remain primary isolation; WRITE_RESTRICTED was additive. Documented waiver in commit body required. |
| T-15-02-02 | Spoofing | direction-b: AppID WFP identity replaces per-session SID; two sessions of same binary share one AppID filter | accept | AppID WFP still kernel-enforced; per-session SID differentiation is a best-effort enhancement, not a security requirement. Documented in commit waiver. |
| T-15-02-03 | Tampering | Pipe-based stdout relay (direction-b) carries unsanitized child output to attach client | accept | No security processing on stdout content; the threat model does not trust child output regardless of transport (same as PTY path). |
| T-15-02-04 | Denial of Service | PTY gate bug suppresses PTY for interactive `nono shell` on Windows | mitigate | `#[cfg(target_os = "windows")]` gate checks `session.interactive_pty` separately from `session.detached_start`; unit test asserts `interactive_pty=true` still allocates PTY. |
| T-15-02-05 | Information Disclosure | `NONO_DETACHED_LAUNCH` env var readable by grandchild (direction-b token gate) | accept | Env var carries no secret; it is a session-mode flag. Grandchild may read it; this is equivalent to the existing `NONO_DETACHED_SESSION_ID` pattern. |
