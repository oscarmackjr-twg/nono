<objective>
Land the highest-risk handle type — Job Object — and the profile schema that
retroactively enables the bind/listen/ReadWrite paths gated in Plans 18-01/18-02.
The Job Object broker is structurally byte-identical to event/mutex (DuplicateHandle
with caller-validated mask) but carries a singular footgun: a profile-widened
JOB_OBJECT_TERMINATE on the supervisor's own containment Job lets the child kill
the supervisor process tree. This plan adds the runtime guard that makes the worst
case STRUCTURALLY IMPOSSIBLE regardless of profile content.

Purpose: Complete the AIPC-01 implementation surface by:
1. Adding the Job Object broker (`broker_job_object_to_process`) + dispatcher arm
   with the containment_job runtime guard.
2. Wiring the profile schema (`capabilities.aipc` JSON object) end-to-end:
   loader (profile/mod.rs), data (policy.json built-in profiles), schema
   (nono-profile.schema.json), and resolved-allowlist plumbing into the dispatcher.
3. Cleaning up the audit-redaction test family by parameterizing it over all 6
   HandleKind shapes (replaces the per-kind tests added in Plans 18-01/18-02).
4. Adding a new Windows-only integration test (`aipc_handle_brokering_integration.rs`)
   that exercises end-to-end broker round-trip for all 5 new handle types using
   `BrokerTargetProcess::current()` (no admin needed).

- 1 new broker function (`broker_job_object_to_process`) + per-fn unit tests

       ```rust
       /// Duplicate a Windows Job Object handle into the target process with
       /// the caller-validated access mask. The mask is pre-validated against
       /// the per-session AIPC allowlist by the dispatcher (Plan 18-03);
       /// additionally, the dispatcher MUST refuse to broker the supervisor's
       /// own `containment_job` HANDLE regardless of mask (CONTEXT.md D-05
       /// footnote runtime guard) — this function trusts that check has
       /// already been performed.
       ///
       /// Per CONTEXT.md D-10: caller closes the supervisor source after
       /// `send_response` returns; this function does NOT close the source.
       pub fn broker_job_object_to_process(
           handle: HANDLE,
           target_process: BrokerTargetProcess,
           mask: u32,
       ) -> Result<ResourceGrant> {
           let mut duplicated: HANDLE = std::ptr::null_mut();
           // SAFETY: `handle` is a live Job Object HANDLE owned by the supervisor.
           // `target_process.raw()` was wrapped from a live process handle.
           // `duplicated` points to writable storage. `mask` was validated by the
           // caller against the per-session allowlist via `policy::mask_is_allowed`.
           // The dispatcher additionally refuses to broker the supervisor's own
           // `containment_job` HANDLE — see `handle_job_object_request` runtime
           // guard. `dwOptions = 0` (NOT DUPLICATE_SAME_ACCESS) — MAP DOWN.
           let ok = unsafe {
               DuplicateHandle(
                   GetCurrentProcess(),
                   handle,
                   target_process.raw(),
                   &mut duplicated,
                   mask,
                   0,
                   0,
               )
           };
           if ok == 0 || duplicated.is_null() {
               return Err(NonoError::SandboxInit(format!(
                   "DuplicateHandle (Job Object, mask=0x{mask:08x}) failed: {}",
                   std::io::Error::last_os_error()
               )));
           }
           Ok(ResourceGrant::duplicated_windows_job_object_handle(
               duplicated as usize as u64, mask,
           ))
       }
       ```

       ```rust
       fn handle_job_object_request(
           request: &nono::CapabilityRequest,
           target_process: nono::BrokerTargetProcess,
           user_session_id: &str,
           runtime_containment_job: windows_sys::Win32::Foundation::HANDLE,
       ) -> Result<Option<nono::supervisor::ResourceGrant>, nono::NonoError> {
           use nono::supervisor::{HandleTarget, HandleKind, policy};
           use windows_sys::Win32::Foundation::CloseHandle;
           use windows_sys::Win32::System::JobObjects::{OpenJobObjectW, JOB_OBJECT_ALL_ACCESS};

           let raw_name = match request.target.as_ref() {
               Some(HandleTarget::JobObjectName { name }) => name,
               _ => return Err(nono::NonoError::SandboxInit(
                   "target shape does not match kind JobObject".to_string()
               )),
           };
           validate_aipc_object_name(raw_name)?;

           // Resolved mask comes from the runtime's pre-resolved AipcResolvedAllowlist
           // (Task 2 below); for now use the hard-coded default. After Task 2, the
           // dispatcher passes the resolved mask in via a runtime field; this helper
           // signature gains an Arc<AipcResolvedAllowlist> parameter at that time.
           let resolved = policy::JOB_OBJECT_DEFAULT_MASK;
           if !policy::mask_is_allowed(HandleKind::JobObject, request.access_mask, resolved) {
               return Err(nono::NonoError::SandboxInit(format!(
                   "access mask 0x{:08x} not in allowlist for JobObject (resolved: 0x{:08x})",
                   request.access_mask, resolved
               )));
           }

           let canonical = format!("Local\\nono-aipc-{}-{}", user_session_id, raw_name);
           let wide: Vec<u16> = std::ffi::OsStr::new(&canonical)
               .encode_wide().chain(std::iter::once(0)).collect();
           // SAFETY: wide is null-terminated UTF-16; OpenJobObjectW returns NULL
           // on failure. We open with FALSE inheritance; mask is JOB_OBJECT_ALL_ACCESS
           // server-side then DuplicateHandle MAPs DOWN to the validated mask.
           let job: HANDLE = unsafe { OpenJobObjectW(JOB_OBJECT_ALL_ACCESS, 0, wide.as_ptr()) };
           if job.is_null() {
               return Err(nono::NonoError::SandboxInit(format!(
                   "OpenJobObjectW(\"{canonical}\") failed: {}",
                   std::io::Error::last_os_error()
               )));
           }

           // CONTEXT.md D-05 footnote runtime guard: refuse to broker the
           // supervisor's own containment_job HANDLE regardless of profile
           // widening or mask. Compare HANDLE values numerically — both came
           // from the same kernel object table.
           if !runtime_containment_job.is_null() && job == runtime_containment_job {
               // SAFETY: `job` is a live HANDLE returned by OpenJobObjectW above.
               unsafe { CloseHandle(job); }
               return Err(nono::NonoError::SandboxInit(
                   "cannot broker the supervisor's own containment Job Object".to_string()
               ));
           }

           let grant = nono::supervisor::socket::broker_job_object_to_process(
               job, target_process, request.access_mask
           );
           // D-10: supervisor closes its source after broker call.
           // SAFETY: `job` is a live HANDLE returned by OpenJobObjectW above.
           unsafe { CloseHandle(job); }
           grant.map(Some)
       }
       ```
