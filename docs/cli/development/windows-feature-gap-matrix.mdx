# Windows Feature Gap Matrix

This matrix tracks the current Windows port state in-repo.

Status labels:

- `working`: available in the Windows preview today
- `partial`: usable in a limited preview-safe form
- `stubbed`: scaffold exists, but the full behavior is not implemented
- `deferred`: intentionally postponed to a later Windows milestone

## Preview Guidance

- Windows preview is intended for startup, config, setup, dry-run, direct command-path validation, and a small directory-only live-run filesystem subset.
- If a command would require real sandbox enforcement, the preview should fail loudly rather than report success.
- Supervisor plumbing is intentionally scaffolded ahead of full supervised execution so later work can attach real capability expansion without redesigning the transport again.
